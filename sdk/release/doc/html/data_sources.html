Data Sources¶

namespace quotek::datasource¶
class datasource¶
#include <datasource.hpp>

datasource is a meta class meant to retrieve data from multiple data sources.

class rss¶
#include <rss.hpp>

rss is a class which allows to fetch rss feeds inside quotek strategies in order to process them (and potentially dedude positions to take).

class yahoofinance¶
#include <yahoofinance.hpp>

yahoofinance is a class which allows to query the Yahoo! Finance API service to fetch historical data on stocks and indices.
