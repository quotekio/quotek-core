History¶

class quotek::data::history¶

history is a container which aims to store historical data of an asset fetched from a remote or local datasource. While quotek::data::record is meant to store raw asset values provided by broker in real-time, history is more about storing historical data having a longer tick, usually the kind of data available from quotes information providers.
